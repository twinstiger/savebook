Speedrunning Basics - Complete Beginner's Guide

Last Updated: May 2026

What is Speedrunning?

Speedrunning (speedrun) is an activity aimed at completing games in the shortest time possible. Players master game mechanics, exploit glitches, and memorize procedures to challenge extreme speeds. The speedrun community is a group full of passion and creativity.

Speedrun Categories

Any%

The most common speedrun category. The goal is to complete the game "by any means" as fast as possible. Any glitches and tricks in the game can be used.

100%

Collect all collectible items, complete all side quests, find all hidden content for a complete speedrun. Usually takes the longest time.

Glitchless

Speedrun without using any program glitches. Suitable for players who want normal gameplay but pursue speed.

Basic Requirements to Start Speedrunning

  • Familiar with the game flow - complete at least once
  • Master basic operations and mechanics
  • Know the target time
  • Download timing software (like LiveSplit)

Common Speedrun Techniques

Sequence Break: Skip game stages that normally need to be completed.

Frame-perfect Operations: Complete actions within precise frame windows to save time.

Clipping/Through-clipping: Use collision detection glitches to traverse terrain.

Recommended Beginner Games

For your first speedrun, choose games with shorter gameplay and higher fault tolerance: Super Mario 64, Portal, and Celeste are excellent choices for beginners.
